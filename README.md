# DATA221 – Assignment 4

Each question is in its own file.

- `DATA221_Assignment3_Question1.py`: Loads the breast cancer dataset, builds `X` and `y`, reports shapes, reports class counts, and answers the class balance questions.
- `DATA221_Assignment3_Question2.py`: Uses an 80/20 split with stratification, trains a Decision Tree with entropy, reports training/test accuracy, and answers entropy/overfitting questions.
- `DATA221_Assignment3_Question3.py`: Trains a constrained Decision Tree, reports training/test accuracy, shows the top five important features, and answers complexity/interpretability questions.
- `DATA221_Assignment3_Question4.py`: Standardizes features, trains a neural network with a sigmoid output unit, reports training/test accuracy, and answers scaling/epoch questions.
- `DATA221_Assignment3_Question5.py`: Uses Question 3 and Question 4 outputs to compute and print confusion matrices for both models.
- `DATA221_Assignment3_Question6.py`: Loads Fashion MNIST, normalizes and reshapes images, trains a CNN for 15 epochs, and reports test loss and test accuracy.
- `DATA221_Assignment3_Question7.py`: Uses the CNN from Question 6 to generate predictions, computes the confusion matrix, and displays three misclassified test images with true and predicted labels.
